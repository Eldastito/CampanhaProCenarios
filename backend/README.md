# Backend

Backend sugerido para o CampanhaPro Cenários.

## Stack inicial
- Python 3.11+
- FastAPI
- Pydantic
- SQLAlchemy
- PostgreSQL
- Redis

## Responsabilidades
- autenticação e autorização
- ingestão do CampanhaPro
- normalização e persistência
- execução de cenários
- previsão e relatórios
- auditoria
