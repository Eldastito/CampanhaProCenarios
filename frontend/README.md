# Frontend

Frontend sugerido para o FORGE Scenario Lab.

## Stack inicial
- React
- TypeScript
- Vite ou Next.js
- Tailwind CSS
- biblioteca de gráficos

## Responsabilidades
- dashboard executivo
- criação e comparação de cenários
- visualização de scores
- relatórios
- auditoria de execuções
